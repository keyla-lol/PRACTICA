# L’Essence – Gastronomic Festival Landing Page

## Description

A modern landing page for a fictional gastronomic festival, designed to showcase elegant culinary experiences.

## Technologies

* HTML5
* CSS3
* JavaScript

## Features

* Semantic structure
* Responsive design (media queries)
* Hamburger menu
* Image gallery with Flexbox
* Event schedule table

## How to Run

Open `index.html` in your browser.
